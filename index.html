id

Описание

Селектор # в html обозначается атрибутом id, должен быть один во всем сайте. Если есть два элемента с одним и тем же id то будут применены стили только к одному который находится ближе к корню дереву.

Пример

target element

class

Описание

target element target

Описание

Стрелка >

Описание

Селектор > выбирает только дочернии элементы (типа selector2) родителя selector1 (на один уровень вложенности)

сигнатура: selector1 > selector2

Пример

target
not target
target
not target

:active

Описание

Селектор :active подсвечивает элемент когда зажата кнопка мыши

сигнатура: :active

Пример

:active 1 :active 2

:focus

Описание

Селектор :focus подсвечивает элемент когда зажата кнопка мыши и если после нажать на таб то будет перемещаться дальше и приминять стили :focus (как видно из примера на div не работает в сравнении с :active)

сигнатура: :focus

Пример

:focus 1 :focus 2

::first-letter

Описание

Селектор ::first-letter выбирает первую букву в родителе независимо от того на сколько глубого вложен текст (кроме inline)

сигнатура: ::first-letter

Пример

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quo, similique?

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Debitis, est.
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Mollitia, quod?
вложенный элемент
текст после вложенного элемента
